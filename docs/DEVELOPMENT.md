## Contributing to C++ Code Quest

### Setting Up Development Environment

1. **Fork the Repository**
   ```bash
   git clone https://github.com/yourusername/cpp-code-quest.git
   cd cpp-code-quest
   ```

2. **Local Development**
# Simple HTTP server
python -m http.server 8000
# Or use Node.js
npx http-server

3. **Open Browser**
http://localhost:8000
